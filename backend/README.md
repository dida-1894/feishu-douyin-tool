## Install

```
git clone https://github.com/Johnserf-Seed/TikTokWeb.git
cd TikTokWeb
npm i
```

## Run

```
npm start
http://localhost:4000/
```

## Docker
```
docker pull johnserfseed/tiktokweb:latest
docker run -d -p 4000:<any port> tiktokweb
```
